1# Hyperlocal Marketplace Monorepo

## Stack
- Mobile apps: Flutter (iOS + Android)
- Backend: Node.js
- Database: Postgres
- Cache/queue: Redis (planned)

## Structure
- `apps/buyer` — Flutter buyer app
- `apps/seller` — Flutter seller app
- `apps/admin` — Admin panel (web)
- `services/api` — Node backend APIs
- `packages/shared` — Shared types/utils/contracts
- `docs` — Architecture, DB schema, API specs
